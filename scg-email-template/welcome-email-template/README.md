# Welcome Email Template - Galleria

A beautiful welcome email template featuring an artistic header design with a textured red background, perfect for showcasing African arts.

## Files

- **`index.html`** - Email-compatible version optimized for email clients
  - Uses table-based layouts for maximum compatibility
  - Inline styles for email client support
  - Simplified gradient background that works across email clients
  - Safe for use in email marketing platforms

- **`preview.html`** - Full-featured preview version for browser viewing
  - Includes all advanced CSS effects (animations, backdrop filters, textured overlays)
  - Perfect for previewing the design before sending
  - Uses external stylesheet (`styles.css`)

- **`styles.css`** - Stylesheet for the preview version
  - Contains all advanced styling including animations
  - Responsive design for mobile devices

## Design Features

### Header Section
- **Background**: Dominant red (#8B0000) with gradient variations
- **Texture**: Abstract brushstroke effects in brighter red, white, and hints of green and brown
- **Oval Overlay**: Semi-transparent rounded oval in the top-right corner (preview version)
- **Logo**: "Galleria" in white serif font with shadow/glow effect
- **Logo Container**: Rounded pill-shaped container with semi-transparent background and white border
- **Welcome Text**: "Welcome to Galleria. A platform to showcase African arts." in white serif font

### Color Palette
- Primary Red: #8B0000 (Dark Red)
- Accent Red: #A52A2A (Brown Red)
- Light Red Overlay: rgba(205, 92, 92, 0.25)
- Text: #ffffff (White)
- Background: #f5f5f5 (Light Gray)

## Usage

### For Email Sending
Use `index.html` - copy the HTML content and paste it into your email marketing platform (Mailchimp, SendGrid, etc.) or send directly via your email service.

### For Preview
Open `preview.html` in a web browser to see the full design with all visual effects.

## Email Client Compatibility

The `index.html` version is optimized for:
- Gmail (web and mobile)
- Outlook (desktop and web)
- Apple Mail
- Yahoo Mail
- Most modern email clients

## Customization

To customize the template:
1. Modify colors by changing the hex values in the inline styles
2. Update text content directly in the HTML
3. Adjust padding and spacing values for different layouts
4. Replace placeholder body content with your actual message

## Notes

- The preview version uses modern CSS features that may not work in all email clients
- Always test the email version (`index.html`) in multiple email clients before sending
- Consider using a background image hosted on a server for more complex textures in production emails
