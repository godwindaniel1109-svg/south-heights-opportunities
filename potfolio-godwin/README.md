## Customization

### Adding Your Profile Image
Replace the profile placeholder in the hero section:
```html
<div class="hero-image">
    <img src="your-profile-image.jpg" alt="Godwin Daniel" class="profile-img">
</div>
```

### Updating Content
- Edit personal information in `index.html`
- Adjust colors and fonts in `styles.css` (CSS variables at the top)
- Modify animations and interactions in `script.js`

### Adding New Sections
1. Add HTML structure in `index.html`
2. Style the section in `styles.css`
3. Add any required JavaScript functionality in `script.js`
